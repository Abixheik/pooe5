INTRODUCTION

Durant ce semestre, nous avons été introduits au concept de la Programmation Orientée Objet (POO). Ce projet a pour but de mettre en œuvre les différentes compétences acquises au cours de ce semestre. Pour cela il nous a été confié le développement d’une application ayant pour but de manipuler des répertoires et fichiers en extrayant les informations pertinentes, en particulier les métadonnées EXIF et XMP pour les images, pour cette fonctionnalité seuls les JPG, PNG et WEBP sont pris en compte pour le moment.

Pour se faire nous proposons deux interfaces :
● Une interface graphique (GUI)
● Une interface console (CLI)

Ces deux interfaces sont compatibles java SE 21, d’un côté le GUI qui propose une interface ergonomique pour les utilisateurs, de l’autre le CLI qui assure le fonctionnement de l’application dans un terminal.
